HTML-Only Security Cam

This is a static WebRTC pair that works with manual copy/paste signaling (no server needed).

1) Open camera.html on the computer with the webcam.
2) Click Start camera, then Create Offer. Copy the Offer text.
3) On your phone, open viewer.html (HTTPS), paste the Offer, click Create Answer.
4) Copy the Answer back into the camera page and click Apply Answer.

Hosting

For camera access, open over HTTPS or localhost. An easy option is GitHub Pages or any static host. Just upload these files and visit the URLs.

Note: On strict networks, you might need a TURN server to connect through firewalls/NATs. This demo uses public STUN only.
