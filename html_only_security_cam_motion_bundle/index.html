HTML-Only Security Cam + Motion Detection

Two static pages. No backend. Use manual copy/paste signaling.

1) Open camera.html on the webcam computer.
2) Click Start camera → Create Offer and copy the JSON.
3) On your phone open viewer.html, paste the Offer → Create Answer.
4) Copy the Answer back into the camera page and click Apply Answer.

Notes

  • Open over HTTPS (GitHub Pages is fine). For local testing use localhost.
  • Motion controls are on the camera page (Enable Motion, sensitivity, pixel delta, interval, cooldown).
  • Strict networks may need a TURN relay; this demo uses STUN only.
